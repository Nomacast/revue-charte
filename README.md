# Revue charte Nomacast — outil d'annotation

Galerie des 9 déclinaisons de charte (A→I) de la page d'accueil nomacast.fr,
avec outil d'annotation pour la graphiste : épingles cliquables + verdict + export.

- Ouvrir : la page GitHub Pages de ce dépôt.
- Les notes sont sauvegardées localement dans le navigateur (localStorage).
- Bouton « Exporter mes annotations » → fichier `.md` à renvoyer.

Images générées depuis les fichiers de test `index-charte-<lettre>.html` (non versionnés ici).
